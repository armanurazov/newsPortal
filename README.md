# newsPortal
A web portal created using APIs to get the latest news from various sources. Deployed on heroku:: https://my-web-portal.herokuapp.com

the project is done using:
- Node.js
- Express.js
- API fetching
- Geolocation library
- CSS Bootstrap grid
